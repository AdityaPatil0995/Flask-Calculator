# Lavender Flask Calculator
A beautiful, sleek, and minimalist web calculator inspired by the modern Windows 11 Standard layout. This project features a completely custom, eye-catching **pastel lavender and deep amethyst** visual theme. 
Unlike basic static frontend

## Features
- **Custom Lavender Aesthetics:** Soothing pastel purple keys contrasted against an elegant dark-plum backdrop with vibrant neon lilac accents.
- **Streamlined Layout:** Completely clutter-free interface with advanced rows removed to prioritize standard, rapid daily calculations.
- **Core Math Operations:** Full support for Addition (`+`), Subtraction (`-`), Multiplication (`×`), and Division (`÷`).
- **Essential Math Actions:** Includes percentage calculations (`%`), sign toggling (`+/-`), clear commands (`C` / `CE`), and smooth backspace tracking (`⌫`).
- **Responsive Animations:** Tactile active states and glowing hover effects engineered strictly using CSS transitions.

## Tech Stack
- **Backend:** Python 3, Flask
- **Frontend:** HTML5, CSS3 (CSS Grid & Flexbox)
## How to Run Locally
Follow these quick steps to download and run this calculator on your machine:
### 1. Clone the Repository
Open your terminal and clone the project:
git clone [https://github.com/AdityaPatil0995/Flask-Calculator.git](https://github.com/AdityaPatil0995/Flask-Calculator.git)
### 2. Move into the Folder
Navigate inside the downloaded directory: cd Flask-Calculator
### 3. Start the Flask Server
Run the application script: python app.py
### 4. Open in Browser
Ctrl + Click the local host link generated in your terminal: [http://127.0.0.1:5000](http://127.0.0.1:5000)
